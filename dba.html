Administrador de banco de dados ou DBA(database administrador)

Oque faz ?

Este profissional é responsável por manter o banco de dados seguro, além de gerenciar, monitorar, efetuar backups, aprimorar a performance, dentre outras tarefas.É uma profissão de alta responsabilidade, pois lida com dados que são importantes para a empresa e deve estar sempre preparado para investigar problemas quando houver e dar suas soluções. Muitas pessoas não gostam de trabalhar sob pressão, mas essa é uma profissão que exige isso. Se algo acontecer, como por exemplo perda de um dado importante, o DBA deverá recuperá-lo o mais rápido possível e, é claro, sem falhas.

Principais conhecimentos

Um DBA precisa desenvolver como características pessoais o raciocínio lógico e a capacidade de trabalhar em equipe, além de buscar por um alto nível de concentração, saber lidar com prazos e trabalhos sob pressão.

Quanto ganha ?

A média salarial para Administrador de Banco de Dados no Brasil é de R$ 7.090,00.
